### PCAP-StoryTeller
*Network Forensics & Attack Visualization*

Transforms raw PCAP files into interactive attack storyboards. Parses TCP/UDP/DNS/HTTP/TLS/ICMP traffic, builds causal attack graphs, scores risk, maps geolocation, and generates forensic reports.

**Stack:** Python · Flask · Scapy · vis.js · Leaflet · ReportLab

**Key features:**
- Multi-protocol PCAP parser with causal event linking
- Interactive attack graph visualization
- Threat detection engine (port scan, data exfiltration, C2 patterns)
- Geolocation intelligence with Folium maps
- PDF/Word/JSON report generation

*Universal Decoder for Security Research*
